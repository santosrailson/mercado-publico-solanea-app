# Frontend - Mercado Público

React + TypeScript + Vite frontend para o sistema de gestão do Mercado Público.

## Tecnologias

- React 18
- TypeScript
- Vite
- Zustand (State Management)
- TanStack Query (React Query)
- Axios
- Recharts (Gráficos)
- Tailwind CSS
- React Router DOM

## Estrutura

```
src/
├── components/     # Componentes reutilizáveis
├── pages/         # Páginas da aplicação
├── services/      # API clients
├── hooks/         # Custom hooks
├── store/         # Zustand stores
├── types/         # TypeScript types
└── utils/         # Utilitários
```

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
